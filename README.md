# 자유규격

제작자가 임의의 위치에 편집하여 업로드 합니다

일부가 변형되거나 잘릴수 있습니다.

위치에 대한 별도 요청이 없을경우 자유규격으로 간주합니다

주로 건물벽에 부착됩니다

특정 구역을 요청하시면 검토후 그곳으로 부착합니다
